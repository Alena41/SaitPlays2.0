# Приложение с играми для развития образования и моторики детей
## Описание - Приложение с двенадцатью играми, которые развивают моторику и образование детей.
  Список игр: 
    1) Математика
    2) Диктант
    3) Виселица
    4) Лабиринт
    5) Английский цвет
    6) Животные
    7) Таракашка
    8) Кликер
    9) Соты
    10) Цвета
    11) Дельфины
    12) Пин-понг
